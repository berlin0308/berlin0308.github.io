The Beginning: Embedded Vision-based Alarm System

Our first major industry project, known simply as the Embedded Vision-based Alarm System, was a comprehensive edge computing solution that monitored retail environments using computer vision and AI optimization technologies. It was targeted at retail loss prevention and was used to demonstrate that embedding optimized AI inference within a real-time monitoring system can drastically increase detection accuracy and system reliability in commercial environments. It was an incredibly promising result that showed that heterogeneous AI acceleration could provide mission-critical performance for retail security applications.

Building Optimized AI Inference Systems for Real-time Anomaly Detection

Inspired by the retail loss caused by self-checkout systems and the potential of edge AI optimization, we developed our comprehensive embedded vision system deployed in retail environments. This system has been featured in multiple industry presentations and has been successfully detecting anomalies in real-time for over 200+ production trials. To read more about the specific technical implementation and performance optimization, please view our presentation and demo materials.

Our embedded vision system aims to improve retail security and loss prevention by providing real-time anomaly detection and behavioral analysis through optimized computer vision and edge computing technologies. The system uses advanced CNN architectures with multi-modal input channels to identify patterns in customer behavior, barcode processing, and suspicious activities, providing store managers with immediate insights into potential security threats. By leveraging Intel CPU/iGPU heterogeneous acceleration and OpenVINO optimization, our system is designed to help retailers prevent losses and ensure store security through the power of optimized AI inference.

Based on the data gathered from our deployment in retail environments, we have been able to show that our embedded vision system achieves 2.7× faster inference and 2.2× end-to-end speedup compared to baseline implementations. Not only that, but we have seen that the system maintains 99% precision with only 2% false alarm rate, which is well above the performance requirements for mission-critical retail applications! Combine that with the fact that our containerized deployment pipeline successfully manages 6-8 device clusters with high-throughput API services, and our embedded vision system has already been able to help retailers prevent losses and maintain security in real-world environments.

The computer vision and AI optimization techniques used within our embedded vision system were developed in collaboration with Advantech Co. Ltd. This collaboration provided us with deep expertise in edge computing hardware, AI acceleration frameworks, and practical retail deployment requirements. Our partners advised us on which optimization techniques are most effective for heterogeneous computing, what types of inference acceleration provide the best performance gains, and how to maximize both system reliability and computational efficiency within the unique context of retail environments.
